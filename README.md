# 题库生成系统

这是一个基于Deepseek大语言模型API的题库生成系统，专为大学教师设计。

## 功能特点

- 支持从电子教材、试题样例等材料中生成各类试题
- 支持多种题型（选择题、填空题、简答题等）
- 提供单个试题生成和批量生成功能
- 自动生成参考答案
- 支持将生成的试题导出为多种格式
- 利用提示学习、工具学习、检索增强等方法提高试题质量

## 安装步骤

1. 克隆仓库
2. 安装依赖：`pip install -r requirements.txt`
3. 创建`.env`文件并添加Deepseek API密钥：`DEEPSEEK_API_KEY=你的API密钥`
4. 运行应用：`python app.py`

## 使用方法

在浏览器中访问 `http://127.0.0.1:5000` 来使用系统。

## 系统结构

- `app.py` - 主应用入口
- `models/` - 数据模型
- `services/` - 核心服务层
- `templates/` - 前端模板
- `static/` - 静态资源
- `utils/` - 工具函数
